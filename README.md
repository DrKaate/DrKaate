# Data analyst
## Описание: в репозитории представлены проекты курса Аналитик данных (Яндекс.Практикум).

## Основные инструменты и навыки, полученные при обучении:
- Умение получать данные - `PostgreSQL` (подзапросы, временные таблицы, оконные функции)
- Обрабатывать и визуализировать данные - `Python` (библиотеки pandas, numpy, matplotlib, seaborn, Plotly, Folium, SciPy и др.), дашборды `tableau`
- Знание `теории вероятности` и `математической статистики`, умение проводить `A/B-тесты` и применять параметрические и непараметрические методы
- `Рассчет продуктовых метрик` (юнит-экономика, активность пользователей, когортный анализ, LTV и др.)
